# Figure Reproduction in Julia

This repository contains Julia-based notebooks that reproduce results and figures from a published study.

The goal of this repository is **reproducibility**: demonstrating how the original figures can be recreated using independent implementations in Julia.

---

## Contents

### Jupyter notebooks (`notebooks/`)
- Bifurcation analysis
- Kepler model simulations
- Catalyst-based models
- Temperature model experiments

### Pluto notebooks (`pluto/`)
- Interactive Julia notebooks used for exploration and visualization
